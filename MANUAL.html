1. Installation

1
Download the extension folder
Unzip smartfill-extension.zip to a permanent location on your PC (e.g. C:\Extensions\SmartFill\). Don't move or delete this folder after installation.
2
Open Edge Extensions page
In Microsoft Edge, go to: edge://extensions or click the ⋯ menu → Extensions → Manage extensions.
3
Enable Developer Mode
Toggle Developer mode ON (bottom-left of the extensions page).
4
Load the extension
Click Load unpacked → select the autofill-extension folder (the one containing manifest.json). SmartFill will appear in your extensions list.
5
Pin to toolbar
Click the puzzle icon (🧩) in Edge's toolbar → click the pin icon next to SmartFill so the ⚡ icon always appears.
Note: Edge may warn "Extensions are turned off" after a restart in some versions. Just go back to edge://extensions and re-enable SmartFill if needed.

2. Creating & Managing Profiles

Profiles store a set of personal information to fill into forms. You can have multiple profiles — for example, one personal and one professional.

Create a new profile

  1. Click the ⚡ SmartFill icon in the Edge toolbar.
  2. On the Fill tab, click the ＋ button next to the profile dropdown.
  3. Enter a name (e.g. "Work", "Personal") and press OK.
  4. You'll be taken to the Profiles tab automatically — fill in your information.
  5. Click 💾 Save Profile.

Edit an existing profile

  1. Open SmartFill → go to Profiles tab.
  2. Select the profile from the dropdown.
  3. Edit any fields you want to change.
  4. Click 💾 Save Profile.

Delete a profile

On the Profiles tab, select the profile and click the 🗑 button. You must always keep at least one profile.

Privacy: All profile data is stored locally in your browser using chrome.storage.local. It is never sent to any server.

3. Filling a Form

1
Navigate to the form
Open the webpage with the form you want to fill. Make sure the form is visible on screen.
2
Open SmartFill
Click the ⚡ icon in the Edge toolbar.
3
Select your profile
Choose the correct profile from the dropdown. The preview shows what will be filled.
4
Click ⚡ Fill This Page
SmartFill scans all visible fields, matches them to your profile, and fills them in. A message shows how many fields were filled.
Tip: SmartFill fires native input events after filling, so React, Vue, and Angular-based forms register the values correctly.

4. Manual Field Mapping

Some websites use unusual field names that SmartFill can't automatically recognise. The Manual Map tab lets you manually assign these fields.

1
Go to Manual Map tab
Open SmartFill → click the Manual Map tab.
2
Click 🔍 Scan Page Fields
SmartFill scans all visible form fields and lists them. Each field shows:
  • ✓ Auto — automatically matched
  • ? Unmatched — needs manual mapping
3
Assign unmapped fields
For each ? Unmatched field, use the "Map to:" dropdown to select which profile field it corresponds to. You can also override auto-matched fields if needed.
4
Save Mappings
Click 💾 Save Mappings. Your mappings are remembered for this site — next time you fill, SmartFill will use them automatically.
Mappings are persistent: Once you map a field, it stays mapped across browser sessions. You only need to do this once per website.

5. How Field Detection Works

SmartFill uses a 4-layer detection system to identify form fields:

#MethodExample
1autocomplete attribute (most reliable)autocomplete="given-name"
2name / id / placeholder / aria-labelname="first_name", placeholder="Your email"
3Associated label text<label for="f1">First name</label>
4Input type hinttype="email", type="tel"

SmartFill never fills type="password" or type="hidden" fields for security reasons.

7. Tips & Troubleshooting

⚡ Fill button says "No matching fields found"

  • Make sure the form is visible and not inside a popup/modal that hasn't opened yet.
  • Use the Manual Map tab to manually assign fields.
  • Some single-page apps render forms after a delay — wait for the form to fully load before clicking Fill.

⚡ Fields get filled but the form doesn't register the values

  • SmartFill fires native input events to trigger React/Vue/Angular reactivity. If a framework still doesn't detect it, try clicking into each field once after filling.

⚡ "Could not reach the page" error

  • SmartFill cannot inject into Edge's internal pages (edge://), the Extensions page, or PDF viewers.
  • Make sure you're on a regular web page (http:// or https://).

⚡ Extension disappeared after restarting Edge

  • Edge sometimes disables unpacked extensions on restart. Go to edge://extensions and re-enable SmartFill.
  • For a permanent installation without this issue, the extension would need to be published to the Microsoft Edge Add-ons store.

⚡ How to update the extension

  1. Replace the files in your autofill-extension folder.
  2. Go to edge://extensions → click the 🔄 refresh button next to SmartFill.
